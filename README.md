# Tapirgo

A simple Ruby library for the [Tapir](http://tapirgo.com/) API.

## Installation

Add this line to your application's Gemfile:

    gem 'tapirgo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tapirgo
